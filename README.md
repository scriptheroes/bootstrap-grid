bootstrap-grid
==============

Do you _just_ want Bootstrap's grid? You can find it here with an entirely self-contained files written in Sass. Have fun.


# Sass
```SCSS
	.example--col-xs-1 {
  		@extend %col-xs-1;
	}

	.example--col-md-2 {
		@extend %col-md-2;
	}

	.example--container {
		@extend %container;
	}

```
